Telegram-бот для отслеживания информации об учебном процессе. Собирает информацию из разных источников и выдает по запросу.

Возможности:
- Полезные ссылки: Доступ к важным ссылкам, связанным с образовательным процессом.
- Различные расписания: Просмотр различных типов расписаний, таких как учебные недели, графики консультаций и т.д.
- Расписание занятий: Получение расписания занятий для определенной группы.
- Расписание звонков: Получение расписания звонков.
- Замены на завтра: Получение уведомлений об изменениях в расписании на следующий день.

Установка и Настройка:
Необходимые Условия:
- Python 3.7+
- Виртуальное окружение (необязательно, но рекомендуется)
- SQLite3
- Требуемые Python пакеты (указаны в requirements.txt)

Шаги Установки:
1. Клонируйте репозиторий:
git clone https://https://github.com/z4knm/tgbot4study
cd tgbot
 
2. Активируйте виртуальное окружение:
venv\Scripts\activate
 
3. Установите необходимые пакеты:
pip install -r requirements.txt
 
4. Настройте бота:
Откройте main.py и поставьте в bot = Bot(token='свой токен')

Запуск бота:
python main.py

Структура Файлов:
- main.py: Точка входа для бота.
- handlers.py: Содержит все обработчики и управление состояниями бота.
- keyboards.py: Определяет клавиатуры, используемые ботом.
- bd.py: Функции для взаимодействия с базой данных.
